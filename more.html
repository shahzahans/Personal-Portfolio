Algorithm Analysis

Algorithm Analysis Strategies

This project demonstrates my understanding of core algorithm design strategies including Greedy Algorithms, Divide and Conquer, Dynamic Programming, and NP Problems.

It includes implementations and explanations of classic algorithms such as Huffman Coding, Fractional Knapsack, Karatsuba Multiplication, Fast Fourier Transform (FFT), Longest Increasing Subsequence (LIS), Edit Distance, and Vertex Cover.

Through this work, I analyzed problem-solving strategies, compared time complexities, and practiced implementing theoretical concepts in Python.
